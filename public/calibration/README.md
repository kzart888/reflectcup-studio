# ReflectCup digital calibration fixtures

These public fixtures test direction, readable detail, spatial frequency and face reconstruction before a physical cup is available.

- `reflection-checker-2048.png`: deterministic 16×16 checker with a unique row-column label in every cell, four corner colors and orientation arrows.
- `frequency-sweep-2048.png`: deterministic frequency and color sweep.
- `text-orientation-2048.png`: deterministic short-text, alphabet, number and direction target.
- `synthetic-portrait.png`: fictional, AI-generated portrait created specifically for this calibration pack; it does not depict or intentionally resemble a real person.

The deterministic fixtures are generated by `pnpm calibration:generate`. The portrait is intentionally not regenerated by that command so releases remain visually stable.
